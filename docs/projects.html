Python to-do using Tkinter library

This project was sincerely confusing, working with Tkinter was one heck of a ride however, I sure did enjoy bringing everything to life and at least get it to work, in hindsight, I realized I did spend a lot of time on it but I sure did learn something, It's a to-do app with a graphical user interface, meaning you can interact with it instead of a command line or terminal, with buttons, entry fields, etc. You register yourself with your name and age, a file is created and you get to logging your to-do's! Every to do and it's history is stored in a file, meaning you can always check the status of your quests at any point in time, completeing a quest is easy, even though there isn't any thorough confirmation if you actually did do the quest, entering the quest's name and clicking on complete changes it's status to done, as easy as that.
